Required Packages

Required packages to run SHARPpy:

  • PySide (for plotting)
  • Numpy (for running SHARPTAB routines)

Both packages can be obtained online or through the conda, easyinstall, or pip package managers.

Installing SHARPpy

To install SHARPpy in your Python directory (using the SHARPy codebase):

  1. Download the SHARPPy package from GitHub by typing the following command in your command prompt:
    git clone http://github.com/sharppy
  2. Unzip the SHARPpy zip file.
  3. cd into the directory and type in:
    python setup.py install
  4. Test the installation by loading up a Python terminal and typing:
    import sharppy
