Credits

SHARPpy was originally written by Patrick Marsh (Storm Prediction Center; SPC) and John Hart (SPC) and is now developed and maintained by the current co-lead developers, Kelton Halbert (OU SoM) and Greg Blumberg (OU SoM/CIMMS).

Special thanks goes to those who have made valuable contributions to the project. Without contributions from many of the individuals listed below, the release of this project would not be possible:

  • Rich Thompson (SPC)
    • Donated code and databases from the original SPC NSHARP to be used in this project.
  • Ryan Jewell (SPC)
    • Provided sounding data for a more rigorous testing of the code base and guidance on implementing SARS.
  • Bryan Smith (SPC)
    • Provided data to make VROT and EF-STP insets.
  • Matthew Bunkers (NWS UNR)
    • Provided guidance in developing a new storm motion vector algorithm for the code and donated the NWS UNL precipitable water vapor climatology.
